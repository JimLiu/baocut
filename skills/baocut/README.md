# baocut skill

The Agent Skill that drives the [BaoCut](https://baocut.app) macOS app from the
command line. See [`SKILL.md`](SKILL.md) for the full agent instructions and
[`references/`](references/) for the per-task guides.

- **Requires:** macOS + the BaoCut app (`bin/baocut` resolves the CLI inside
  `/Applications/BaoCut.app`).
- **Install:** `npx skills add JimLiu/baocut` — see the [repo README](../../README.md).
